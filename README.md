# Number Analyzer (Python)

A simple Python project that analyzes a list of numbers.

## Features
- Count total numbers
- Calculate sum
- Find maximum and minimum
- Calculate average
- Count odd and even numbers
- Find second largest number
- Check if list is sorted

## Concepts Used
- Python functions
- Lists
- Loops
- Conditional statements
